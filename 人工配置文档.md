三 兼容 reader.py 的可识别字段（其余字段不处理）

1) 顶层字段
- date -> device_timestamp(统一毫秒)
- project_name
- project_num
- dev_name
- dev_num

2) datas 中仅识别以下键名（必须是不带前缀的键名）
- SNR -> signal_snr
- RSRP -> signal_rsrp

3) Wavevalue[0] 中可识别键名
- Atemp -> a_temp
- Btemp -> b_temp
- Ctemp -> c_temp
- Htemp -> in_temp
- Hdamp -> in_hum
- PhaseTempA -> a_temp
- PhaseTempB -> b_temp
- PhaseTempC -> c_temp
- InSideTemp -> in_temp
- InSideHumi -> in_hum
- OutSideTemp -> out_temp
- OutSideHumi -> out_hum
- MlxMaxTemp -> mlx_max_temp
- MlxMinTemp -> mlx_min_temp
- MlxAvgTemp -> mlx_avg_temp
